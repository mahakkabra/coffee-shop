# ☕ Coffee Shop Website (Frontend)

A modern and responsive coffee shop website UI built using HTML, CSS, and basic frontend design principles. This project showcases a clean landing page with product listings, customer reviews, and brand storytelling.


## Features
- Responsive navigation bar
- Hero section with call-to-action
- About / History section
- Product listing with pricing
- Customer reviews section
- Footer with contact details
- Clean and consistent UI design


## Tech Stack
- HTML5
- CSS3
- JavaScript
